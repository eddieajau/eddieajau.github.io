[**Engineer**][0] version 2.0.4 is now available for download.  This version fixes several small issues reported by users.  For more information see the [**Engineer**][0] page at the Art of Joomla web site.  **Engineer** is a Joomla Administrator component that assists component developers in building XML installation files for their extensions.


Engineer is now being supported directly by New Life in IT from [The Art of Joomla][1] web site (formerly available from [jxtended.com][2]).  The move has been made to make this extension more available to community for download and further contribution.  The source code to this extension is now freely available.  Current improvements to the component include the ability to build various files for MVC component.
