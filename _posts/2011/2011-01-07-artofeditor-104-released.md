ArtofEditor Version 1.0.4, a WYSIWYG editor plugin for Joomla 1.5 and 1.6 based on the very popular CKEditor, has now been released.  This version includes updates to the latest version of CKEditor 3.5\.

For more information and download details see the [extension page for ArtofEditor][0].

For more information on CKEditor 3.5 see the [vendor's release information][1].

New features/fixes in this version of the extension include the following:

* Fixed buttons bug in Joomla 1.6 plugin.
