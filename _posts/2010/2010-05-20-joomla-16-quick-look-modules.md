Joomla 1.6 includes several new features for modules.  You can time a module to be published between two dates, and you can set modules to be excluded from particular pages.  Watch the Joomla 1.6 video on modules to find out more.

This is a quick look at two new features in the Joomla 1.6 Module Manager.

[![screenshot](/images/2010/joomla_edit_module1.jpg)][3]

When editing a module, you can now set the _Publish Start_ and _Publish Finish_ dates just as you would for articles.

[![screenshot](/images/2010/joomla_edit_module2.jpg)][4]

The module assignment widget has been improved dramatically.  You can select menu pages by checkboxes (no more accidents that wipe out all your selections), you can choose to show the module on all pages except those selected, and you can toggle the selections in the entire list.

Other features you will notice are the ability to give the module an advisory note and the ability to set the language that this module applies to.
