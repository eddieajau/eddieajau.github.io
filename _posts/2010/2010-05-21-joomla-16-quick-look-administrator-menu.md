Here is an overview of the new format of the Joomla 1.6 Administrator Menu to help you find what you are looking for.

* Control Panel
* Site Maintenance
  * Global Check-in
  * Clear Cache
  * Purge Expired Cache
* System Information
* Logout

### Users

* User Manager
* Groups
* Access Levels
* Add New User
* Add New Group
* Add New Access Level
* Mass Mail Users

* Add New Article
* Add New Category
* Media Manager

* Banners

  * Clients
  * Tracks
  * Categories
* Contacts
  * Contacts
  * Categories
* Messaging
  * New Private Message
  * Read Private Messages
* Newsfeeds
  * Feeds
  * Categories
* Search
* Weblinks
  * Links
  * Categories
* Redirect

* Extension Manager
* Module Manager
* Plug-in Manager
* Template Manager
* Language Manager
