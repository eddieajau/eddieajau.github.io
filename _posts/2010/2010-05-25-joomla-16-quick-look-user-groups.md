This is a quick look at the new user groups feature in Joomla 1.6\.

You can access the user groups manager by selecting _Users -\> Groups_ from the Administrator menubar, or by clicking on the _Groups_ submenu link while in the User Manager.

[![screenshot](/images/2010/joomla_user_groups.jpg)][3]

Add a new user group by clicking the _New_ button in the toolbar.  Type in the name of the group (Joomla doesn't care what you call a user group) and then select the parent group under which you want this group to appear.  Choosing the right parent is important as the permissions inherit down the user group tree (more on this in a future video).

[![screenshot](/images/2010/joomla_group.jpg)][4]

When editing a user, you have the ability to assign the user to one or multiple user groups.

[![screenshot](/images/2010/jooma_edit_user.jpg)][5]

The user manager will show the group or groups that the user is assigned to.

[![screenshot](/images/2010/joomla_users.jpg)][6]
