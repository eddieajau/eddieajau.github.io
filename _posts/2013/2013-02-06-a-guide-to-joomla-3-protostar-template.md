The guide currently includes information about the following aspects of the template:

* The module positions in Protostar
* The custom module styles (chrome) that come with Protostar
* The nav-pills, nav-list and nav-tabs menu styles.
* The error page (`templates/protostar/error.php`)
* The component (popup / modal) page (`templates/protostar/component.php`)
