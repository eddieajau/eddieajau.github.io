This is an Eclipse snippet category with code snippets to assist with developing backend edit pages in Joomla 2.5 components.


It includes the following snippets:

* Backend Item Sub-controller
* Backend Item Model
* Backend Item XML Form
* Backend Item Edit View
* Backend Item Edit Layout
* Backend Item Edit Params Sub-layout
* Backend Item Edit Metadata Sub-layout
* Backend Item Edit INI Strings

Watch **[this video][0]** to learn how to use Eclipse snippets.

Note that many of the snippets will be applicable for Joomla 3.0\. The major differences will be in the markup of view layout files.

**[Download][1]** this snippet and import it into your Eclipse Snippets view.
